# MushcFlow - A simple music player!🎵

This project is created with the intention of learning about C++ GUI (QtCreator)📚
#

## What MushcFlow can do?🔨
MushcFlow is a simple music player, but have some basic nice functionalities like:

Download new songs inside the app (with real-time update)!⬇️

You can see the thumbnail of your song!📷

You can change the speed of song!🕰️

Random track and loop!📻

A Log System who will show you if a error occurred! (the folder with logs is in the root folder)📢

You can also add songs who you already have in your device!

You can change the Songs folder! (need to open "configs.json" file in MushcFlow root folder!)

## Support
System: Only available in Windows

Song format: mp3

Thumbnail format: jpg

## Compiling/Installantion 
⚠️ WARNING! If you want to change something in the source code, I strongly recommend using the app QtCreator version 6+!

Compiling: you can just clone or download the source code and open with QtCreator

Installation: you can use the file "MushcFlow_Installer.exe" or if you don`t want to use the installer, download the "MushcFlow.zip" and open "MushcFlow.exe"! (to use the installer, you need to open it with ADM!)

## Help
If your app don`t open or another error occure to you, please go to MushcFlow root folder and delete the file "configs.json"! and open MushcFlow again

The "configs.json" is not responsable for your songs, this file is only to MushcFlow know where is Song Folder

## Libraries used
Qt (for GUI and code)

Yt-dlp (for download/conversion)

Ffmpeg (for yt-dlp)
